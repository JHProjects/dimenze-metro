About The project
This project was made as a part of school work. The task was to make a set of letter stamps along with complete First Day Cover containing steel stamp design and additional press design. All inspired by the interior of Prague metro.
I used some of the design format to make this poster.
The metro interior was made by designer Jaroslav Otruba together with the main metro graphic designer Jiří Rathuský. Using convex or concave aluminum extrusions – each station has a different symbolic design. An iconic and timeless op–artistic design.
